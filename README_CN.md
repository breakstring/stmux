
# Remote Tmux Workflow (RTW)

[English](README.md) | [中文版]

一套专为 Mac 用户优化的远程服务器开发工具集。通过自定义的 tmux 配置和自动化同步脚本，为您提供极致流畅的远程会话管理体验。

## 🚀 核心功能

### 1. 增强版 `.tmux.conf`
* **双前缀支持**：同时支持 `Ctrl + b` 和 `Ctrl + a`，适应不同的肌肉记忆。
* **Mac 友好分屏**：使用 `\` 进行左右分屏，使用 `-` 进行上下分屏。无需再为按 `Shift` 键而手忙脚乱。
* **触摸板优化**：完整开启鼠标/触摸板支持，可通过双指手势平滑滚动查看历史记录。
* **视觉焦点**：采用经典绿色状态栏，当前活跃面板（Pane）使用 **亮红色边框** 突出显示，让您一眼定位光标。
* **持久高亮复制**：优化了鼠标选中文本的逻辑，选中后文字保持高亮，不会立即消失，方便复制操作。

### 2. `stmux` 自动化脚本
* **配置同步**：支持一键将本地 `.tmux.conf` 实时同步至任何远程服务器。
* **智能会话管理**：自动接入已有会话或创建新会话，避免产生重复、混乱的后台进程。
* **脱离不掉线**：在按下脱离快捷键（`Ctrl+a, d`）后，脚本会提示是否继续留在远程 Shell 中，而非直接断开连接。
* **彩色终端输出**：关键操作步骤均有彩色反馈，过程清晰直观。

## 📦 安装方法

### 自动安装（推荐）
克隆本仓库后，直接运行安装脚本：
```bash
chmod +x install.sh
./install.sh

```

### 手动安装

1. 将 `.tmux.conf` 复制到您的Home目录：
```bash
cp .tmux.conf ~/.tmux.conf

```


2. 将 `stmux` 脚本移动到系统路径并赋予执行权限：
```bash
sudo cp stmux /usr/local/bin/stmux
sudo chmod +x /usr/local/bin/stmux

```

## 🛠 使用说明

`stmux` 命令的基本语法如下：
`stmux <ssh-alias> [session_name] [sync]`

* **快速连接**：`stmux my-server`
*(连接到 'my-server' 上的默认会话 'main')*
* **特定任务**：`stmux my-server dev-logs`
*(连接或创建一个名为 'dev-logs' 的会话)*
* **同步并登录**：`stmux my-server main sync`
*(在进入会话前，先将本地最新配置推送至服务器)*

## ⚠️ 前置要求

* **SSH 配置**：脚本依赖于您在 `~/.ssh/config` 中定义的 SSH 别名（Host）。
* **SSH 密钥**：强烈建议配置基于密钥的免密登录，以获得无需反复输入密码的丝滑体验。
